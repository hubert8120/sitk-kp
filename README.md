# SITK RP Oddział w Koszalinie

Projekt strony internetowej Stowarzyszenia Inżynierów i Techników Komunikacji Rzeczpospolitej Polskiej – Oddział w Koszalinie.

## Instalacja

Sklonuj repozytorium, a następnie zainstaluj zależności i uruchom serwer deweloperski:

```sh
npm install
npm run dev
```

Serwer zostanie uruchomiony na porcie `8080`.

## Budowanie aplikacji

Aby zbudować aplikację na środowisko produkcyjne, uruchom:

```sh
npm run build
```
